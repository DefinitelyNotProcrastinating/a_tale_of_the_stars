### 第一部分：核心经济原则

#### 货币体系
-   通用货币: 银河信用点 (Credit, 简称CR) 是全星系公认的价值衡量与交换媒介。所有跨星际贸易的最终结算均通过CR进行。
-   发行与监管: 信用点并非由某一中央银行发行，而是一种基于古老星门网络协议的去中心化数字货币。其总量的增长与星系内的总经济活动（主要是能源开采与物质转换）挂钩，确保了其长期价值的相对稳定。

#### 市场调节机制
-   国家价格乘数: 尽管CR是通用标准，但各势力内部市场会根据其国策、资源丰度、工业水平和意识形态，对标准价格应用一个“国家价格乘数”。这导致同一种商品在不同势力的售价差异巨大。ICE作为中立的贸易平台，通常被视为基准（乘数为1.0）。
-   计算公式: `实际售价 = 标准价格 × 国家价格乘数`

#### 2. 弹药

弹药价格基于其技术含量和破坏能级，而非口径。使用低阶弹药会大幅降低武器性能。

| 弹药技术位阶 (Tier) | 1000发标准弹药价格 (CR) | 核心特性 |
| :--- | :--- | :--- |
| T1 (常规实弹/能量包) | ~50,000 CR | 标准铁壳弹或低功率能量电池，主要用于治安和民用自卫。 |
| T2 (军用级弹药) | ~250,000 CR | 穿甲弹、高爆弹或标准军用等离子包，能有效对付T2级别的护甲。 |
| T3 (精工弹药) | ~1,000,000 CR | 包含少量稀有同位素或经过能量矩阵强化的弹药，开始出现特殊效果。 |
| T4 (史诗级弹药) | ~2,000,000 CR | 采用超空间、灵能或亚稳定奇异物质技术，能对T4目标造成显著威胁。 |
| T5 (传说级弹药) | ~25,000,000 CR | 每一发都蕴含着微型法则造物，如短时奇点或凋零灵气，对主力舰有效。 |
| T6 (神话级弹药) | ~500,000,000 CR+ | 法则级武器的弹药，每一发都可能是战略级效果的载体，通常按“发”计价。 |
| 超级武器 | 不适用 | 超级武器不消耗传统弹药，其开火成本体现为巨量的能源和漫长的冷却时间。 |
