# 技能系统核心框架

本节将定义所有技能的生成、计算与获取方式，以确保其与核心世界观及数值体系的深度整合。

---

## 第一部分：技能通用规则

### 1. 技能生成格式 (强制)

所有技能，无论是角色固有、装备赋予还是载具模块，都必须遵循以下格式进行定义。

```yaml
技能名称: "字符串"
品质: "T1 - T6之一"
类型: "主动/被动"
消耗: "XXX 能量" # 若无消耗，则为"无"
目标类型: "单体 / 范围:X米 / 自身 / 环境"
核心功能: "伤害 / 治疗 / 护盾 / 控制 / 增益 / 减益 / 功能"
依赖项:
  # 描述技能效果的计算权重，总和必须为1.0，格式为 “依赖项:依赖值, 依赖项2:依赖值2……”
  # 依赖项可以是五维属性或六大领域技能
  - 来源: "工程/炮术/科学/灵能/控制/侦测"
    权重: "浮点数 (0.0 - 1.0)"
  - 来源: "..."
    权重: "..."
效果(effects):
  # 描述技能的具体效果，必须包含数值
  - 效果A: "效果描述，如：造成XXX点动能伤害。"
  - 效果B: "..."
描述(details): "关于此技能的背景故事、视觉效果或机制的叙事性文字。"
```

### 2. 伤害与效果类型

技能可以造成以下一种或多种类型的伤害/效果，混合类型需在效果描述中注明。

*   动能: 物理冲击、穿刺、挥砍、磁轨炮。
*   热能: 等离子、激光、火焰。
*   电磁: 粒子束、电弧。
*   奥术: 灵能、仙术、生命力/凋零等超自然能量。
*   精神: 直接作用于心智、意志的攻击。
*   真实: 绕过常规防御的法则级效果。
还有其他属性的伤害……

## 第二部分：技能威力计算

技能的最终效果取决于其基础威力与位阶系数 (Kt)。

### 1. 角色技能计算 (本体 & 个人装备)

角色的技能威力与其个人能力深度绑定。

公式:
`最终效果值 = ( ( Σ ( (依赖项数值) × 对应权重 ) ) + 技能基础威力 )`

*   依赖项数值: 指角色面板上对应的领域等级（如 `灵能: 20`）。
*   对应权重: 定义于技能的 `依赖项` 字段。
*   技能基础威力: 一个固定的数值，由技能的 品质 决定。详见下方的 【技能基础威力参考表】。

### 2. 载具技能计算 (舰船模块)

载具的技能（武器、电子战系统等）伤害独立于驾驶员的个人五维属性，伤害仅与模块本身有关，有效性与领域技能和模块有关。载具技能不受位阶系数 (Kt) 影响。

公式:
`最终伤害值 = 技能基础威力`
*   载具武器不会因为驾驶员的属性而增加伤害，但可以通过驾驶员的领域技能来提升有效值，让它命中率更高或者打出暴击。

## 第三部分：技能基础威力参考表

此表定义了不同品质和功能的技能所具有的 “基础威力” 数值。此数值是计算最终效果的核心基石。

| 品质 | 伤害| 治疗 | 护盾  | 核心特性 |
| :--- | :--- | :--- | :--- | :--- |
| T1 | 10 | 15 | 20 | 单一基础效果。 |
| T2 | 20 | 30 | 50 | 效果增强，可能附带微弱（1回合）的次级效果。 |
| T3 | 50 | 75 | 150 | 标准效果，并可能提供少量战斗修正（如`+5%`暴击率）。 |
| T4 | 250 | 375 | 750 | 效果强大，开始出现改变机制的词条（如“穿透20%护盾”）。 |
| T5 | 1,250 | 1,875 | 3,750 | 拥有改变战局的强力效果（如“2回合群体增伤`+10%`”）。 |
| T6 | 6,250 | 9,375 | 18,750 | 具备法则级能力（如“本次攻击附加真实伤害”或“免疫一次致命伤”）。 |
| T0 | *变量* | *变量* | *变量* | 效果独一无二，不循此规则，由具体设定决定。 |

*注：此处的伤害值为角色技能的基准。最终伤害会乘以位阶系数Kt，从而与《核心数值总表》中的“单次行动预期伤害”范围匹配。*

## 第四部分：技能获取途径

技能的获取主要通过以下两种方式：

1.  学习:
    *   来源: 技能书籍、导师传授、数据芯片。
    *   机制: 通过消耗时间和资源进行学习，可以直接获得一个完整的技能。所获技能的品质、效果和消耗与来源完全一致。

2.  自创:
    *   来源: 经历重大剧情事件、战斗中突破极限、对某个领域的研究达到瓶颈后产生顿悟。
    - 机制: 这是一种自发产生的过程，自创的技能往往更具个人特色。
