### **星战与全域战斗处理系统 (综合协议)**

**核心指令:** 在任何需要战斗的场景时，必须严格遵循本协议进行物理级、无主角光环的并行战斗演算，不要偏袒{{user}}，{{user}}会死，会失败。战斗不再是回合制，而是划分为战术时间切片（dt，1dt ≈ 5-10秒）。所有单位在一个dt内的开火、移动、电子战均**同时发生**，谁先被摧毁取决于伤害结算与物理规则。

#### **【核心战斗与物理逻辑】**
1. **伤害与防御机制:**
   * **舰船载具三血条:** 护盾 -> 装甲 -> 结构。永远先扣除护盾，最后攻击结构。只有攻击结构时才会造成部位破坏。永远不会穿透护盾攻击到装甲，必须完全清空护盾才能攻击装甲，因为护盾结构会自我重分配到全舰。
   * **角色HP:** 角色适用标准的生命值和能量机制。
2. **武器毁伤与命中物理定理:**
   * **能量/光束武器:**  [目标角速度]。【机动补正】：高角速度将会适量减少5 - 15的有效值，若目标角速度低，增加15-30 攻击者命中有效值。
   * **导弹/鱼雷:** 命中必定造成伤害，但基于球形体积重合度计算，长轴越长，受到的伤害越多。公式：`导弹速度系数 = 导弹爆炸速度 / 目标飞行速度 (最大为1)`。目标越小、速度越快，大型导弹对其造成的伤害越低。
   * **重力波武器:** 对更大的目标伤害更大，舰体大小乘数最高为250%伤害。
   * **超光速实体武器:** 对于曲速大炮的动能炮弹、导弹、无人机，若目标距离 > 1000km，将通过超空间跳跃至目标附近 10000km 距离处或者进入杀伤距离，再点亮推进器突防。
   * **电子战攻防:** 对于不依赖某个原理的侦测系统，无法使用该原理来干扰该侦测系统，其ECM **自动失败**。例如，基于雷达/快子束的ECM系统对时空曲率传感器并没有任何作用。同理可得，光学隐身面对时空曲率传感器也是无效的，而时空曲率传感器无法感知目标的颜色。
3. **有效值 (有效值) 与位阶 (Tier) 碾压:**
   * **有效值 计算:** 
     * *角色:* 关联属性 + 技能等级 + 装备加值。
     * *舰船:* 领域/驾驶技能等级 + 舰船系统被动加值（飞船战不计入角色基础五维）。
   * **伤害乘区:** 
     * *初始伤害:* 初始伤害 = (属性+技能+武器)。
     * *舰船/载具伤害:* 舰炮实际伤害 = 武器基础面板 × 炮塔/发射器数量。例如一个武器为8x 800伤害那就造成总计6400伤害。
   * **对抗态势判定 Diff = (攻方有效值 - 守方有效值) / 守方有效值 :**
     * **|Diff| < 5% (势均力敌):** 引入D20判定。D20 + 攻方修正 vs D20 + 守方修正。D20=20必暴击。无论如何，不要忘记写出双方的有效值运算过程。
     * **5% < Diff < 50% (优势对抗):** 攻方具有[优势]（双骰取大或固定命中率补正）。
     * **Diff > 50% (技术压制):** 跳过投掷，无视闪避与常规防御，必定暴击（最大伤害倍率）。若绝对碾压，最终伤害×1.5。
